## 2. 模块职责划分

### 2.1 serai-collector（串口采集模块）

#### 核心职责
- **串口设备管理**: 初始化、打开、关闭串口连接
- **Modbus-RTU协议实现**: 构建Modbus请求、解析Modbus响应
- **数据采集调度**: 按1秒间隔定时采集温湿度数据
- **数据缓存**: 断网时缓存数据，网络恢复后批量上传
- **TCP客户端**: 维持与rtu-gateway的TCP长连接
- **二进制协议封装**: 将采集数据封装为自定义二进制帧

#### 功能边界
✅ **负责**:
- 串口通信的所有细节（波特率、数据位、校验位等）
- Modbus-RTU协议的CRC校验
- 采集任务的定时调度
- 本地数据缓存和重传机制
- 与rtu-gateway的TCP连接管理

❌ **不负责**:
- 数据的业务逻辑处理
- 数据持久化到数据库
- MQTT消息发布
- Web接口提供

#### 关键类设计

### 2.2 rtu-gateway（RTU网关模块）

#### 核心职责
- **TCP服务端**: 接收多个serai-collector的连接
- **二进制协议解析**: 解析自定义二进制帧，提取传感器数据
- **Modbus数据解析**: 解析Modbus响应中的温湿度值
- **数据验证**: 校验数据完整性和合法性
- **MQTT客户端**: 将数据以JSON格式发布到MQTT Broker
- **连接管理**: 管理多个采集器的连接状态

#### 功能边界
✅ **负责**:
- TCP服务端的启动和连接管理
- 二进制帧的解码和验证
- Modbus数据字段的解析（功能码、寄存器值）
- 数据格式转换（二进制 → JSON）
- MQTT消息的发布
- 连接状态监控和日志记录

❌ **不负责**:
- 串口通信
- 数据采集调度
- 数据持久化
- 业务逻辑处理（报警、统计等）
- Web接口提供

#### 关键类设计

### 2.3 web-server（业务服务模块）

#### 核心职责
- **MQTT订阅**: 订阅rtu-gateway发布的数据主题
- **数据持久化**: 将接收到的数据存储到MySQL数据库
- **REST API**: 提供HTTP接口供前端调用
- **RTU管理**: RTU设备的注册、配置、状态管理
- **数据查询**: 历史数据查询、统计分析
- **报警处理**: 阈值判断、报警记录、报警推送
- **配置下发**: 通过MQTT向rtu-gateway下发配置指令

#### 功能边界
✅ **负责**:
- MQTT消息订阅和处理
- 数据库的增删改查
- RESTful API设计和实现
- 业务逻辑处理（报警、统计、权限等）
- 前端页面渲染（如使用Thymeleaf）
- 配置管理和下发

❌ **不负责**:
- 串口通信
- Modbus协议处理
- TCP连接管理
- 数据采集调度

## 3. 内部通信接口

### 3.1 serai-collector ↔ 温湿度传感器（Modbus-RTU）

#### 协议类型
Modbus-n
#### 请求格式（读取保持寄存器 0x03）
```
字节索引  | 字段         | 示例值  | 说明
---------|-------------|---------|------------------
0        | 从站地址     | 0x01    | 设备地址
1        | 功能码       | 0x03    | 读保持寄存器
2-3      | 起始地址     | 0x0000  | 寄存器起始地址
4-5      | 寄存器数量   | 0x0002  | 读取2个寄存器
6-7      | CRC校验      | 0xXXXX  | CRC16校验码
```

#### 响应格式
```
字节索引  | 字段         | 示例值  | 说明
---------|-------------|---------|------------------
0        | 从站地址     | 0x01    | 设备地址
1        | 功能码       | 0x03    | 读保持寄存器
2        | 字节数       | 0x04    | 后续数据字节数
3-4      | 温度值       | 0x00FA  | 温度 = 250 / 10 = 25.0°C
5-6      | 湿度值       |  = 500 / 10 = 50.0%
7-8      | CRC校验      | 0xXXXX  | CRC16校验码
```

#### 数据解析规则
- **温度**: `(高字节 << 8 | 低字节) / 10.0`
- **湿度**: `(高字节 << 8 | 低字节) / 10.0`

### 3.2 serai-collector ↔ rtu-gateway（自定义二进制协议）

#### 协议类型
TCP长连接 + 自定义二进制帧

#### 数据帧格式
```
字节索引  | 字段         | 长度    | 示例值      | 说明
---------|-------------|---------|------------|------------------
0-1      | 帧头         | 2字节   | 0xAA55     | 固定帧头
2        | 版本号       | 1字节   | 0x01       | 协议版本
3        | 帧类型       | 1字节   | 0x01       | 0x01=数据上报
4-5      | 数据长度     | 2字节   | 0x0020     | 后续数据长度
6-21     | RTU ID      | 16字节  | "RTU001..."| RTU唯一标识   | 时间戳       | 8字节   | 毫秒时间戳  | 采集时间
30-N     | Modbus原始数据| 变长    | 0x01030... | Modbus响应帧
N+1-N+2  | CRC16校验    | 2字节   | 0xXXXX     | 整帧CRC校验
```

#### 帧类型定义
```java
public interface FrameType {
    byte DATA_REPORT = 0x01;      // 数据上报
    byte HEARTBEAT = 0x02;        // 心跳包
    byte CONFIG_ACK = 0x03;       // 配置确认
    byte ERROR_REPORT = 0x04;     // 错误上报
}
```

#### 示例数据
```
AA 55 01 01 00 20
52 54 55 30 30 31 00 00 00 00 00 00 00 00 00 00  // RTU001
00 00 01 8E 3F 2A 5C 80                          // 时间戳
01 03 04 00 FA 01 F4 XX XX                       // Modbus数据
XX XX                                             // CRC
```

## 5. 数据模型定义

### 5.1 RTU网关实体
```java
@Data
@Table(name = "rtu_gateway")
public class RtuGateway {
    private Long id;
    private String rtuId;           // RTU唯一标识
    private String name;            // RTU名称
    private String location;        // 安装位置
    private String status;          // 状态: online/offline
    private String serialPort;      // 串口号
    private Integer baudRate;       // 波特率
    private Integer deviceAddress;  // Modbus设备地址
    private Integer samplingInterval; // 采样间隔（秒）
    private LocalDateTime lastOnlineTime;
    private LocalDateTime createTime;
    private LocalDateTime updateTime;
}
```

### 5.2 RTU数据实体
```java
@Data
@Table(name = "rtu_data")
public class RtuData {
    private Long id;
    private String rtuId;
    private Float temperature;      // 温度
    private Float humidity;         // 湿度
    private LocalDateTime timestamp; // 采集时间
    private String rawModbus;       // Modbus原始数据
    private LocalDateTime createTime;
}
```

### 5.3 RTU配置实体
```java
@Data
@Table(name = "rtu_config")
public class RtuConfig {
    private Long id;
    private String rtuId;
    private Integer samplingInterval;
    private Float tempMin;
    private Float tempMax;
    private Float humidityMin;
    private Float humidityMax;
    private Boolean alarmEnabled;
    private LocalDateTime updateTime;
}
```

### 5.4 RTU报警实体
```java
@Data
@Table(name = "rtu_alarm")
public class RtuAlarm {
    private Long id;
    private String rtuId;
    private String alarmType;       // temperature_high/tempeture_low/humidity_high/humidity_low
    private String alarmLevel;      // info/warning/error
    private Float currentValue;
    private Float thresholdValue;
    private LocalDateTime alarmTime;
    private String status;          // unhandled/handled
    private String handleResult;
    private String handler;
    private LocalDateTime handleTime;
}
```
